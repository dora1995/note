涉及到交换的，可以看看双指针是否能满足；双指针法（快慢指针法）在数组和链表的操作中是非常常见的，很多考察数组、链表、字符串等操作的面试题，都使用双指针法

```
var removeElement = function(nums, val) {
    if (!nums || nums.length == 0) {
        return 0
    }
    let head = 0

    for (let i = 0; i< nums.length; i++) {
        const item = nums[i]
        if (item !== val) {  
            if (head !== i) {
                nums[head] = nums[i]
                head
            }
            head ++
        }
    }
    return head
};
```
双指针的话要先模拟一下，找到其中的规律，再编码就容易较多
